Generate Solid Color Favicon

Use this tool to generate a solid color favicon for the Opula website.

Instructions:

  1. Choose a color using the color picker (default is the Opula primary color #2A2A28)
  2. Click "Generate Favicon" to create the favicon
  3. Click "Download Favicon" to download the favicon
  4. Replace the existing favicon.ico in the public folder with this new file
